# BuildX-Analytics-KeerthanaSanthosh
## Project Overview

This project demonstrates an end-to-end data analytics workflow using Python, SQL, and Power BI. The dataset was cleaned, analyzed, and visualized to generate business insights.

## Dataset Selected

Sample Sales Dataset

## Why I Selected This Dataset

I selected this dataset because it contains useful business information such as sales, products, customers, and profit. It provides opportunities to perform data cleaning, SQL analysis, and dashboard creation while answering real business questions.

## Business Questions Answered

1. Which products generated the highest sales?
2. Which regions contributed the highest revenue?
3. What are the monthly sales and profit trends?

## Tools Used

- Python (Google collab)
- Pandas
- SQL
- Power BI
- GitHub

## Key Insight

The analysis showed that a small number of products contributed a significant portion of total sales, while certain regions consistently generated higher revenue. These insights can help businesses focus on profitable products and regions.

## Files Included

- Notebook.ipynb – Data cleaning and analysis
- cleaned_dataset.csv – Cleaned dataset
- queries.sql – SQL queries
- dashboard.pbix – Power BI dashboard
- README.md – Project documentation
